## More Reactions
More Reactions is an addon to [Destroy Mod](https://github.com/Petrolpark-Mods/Destroy/) which adds more reactions.

Feel free to fork or contribute!

To build from repository:
1) Ensure you have a */libs/* folder with destroy mod and petrolpark library jars.
2) Run the gradle jar command
3) The jar is built under */build/libs/*
